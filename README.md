# Free Landingpage Digital Invitation

📄 **Free HTML Landing Page untuk Undangan Digital**

Repository ini berisi template landing page gratis berbasis HTML dan CSS yang bisa kamu gunakan untuk kebutuhan undangan digital.

---

## 🚀 Tutorial

Ikuti langkah-langkah berikut untuk menggunakan dan men-deploy landing page ini secara gratis.

### 1. Buat Akun GitHub

Jika kamu belum memiliki akun GitHub, silakan daftar terlebih dahulu melalui link berikut:

👉 [https://github.com/join](https://github.com/join)

---

### 2. Fork Repository

Fork repository ini ke akun GitHub kamu:

👉 [https://github.com/bangabudesign/free-landingpage-digital-invitation](https://github.com/bangabudesign/free-landingpage-digital-invitation)

Setelah di-*fork*, kamu akan memiliki salinan project ini di akun GitHub kamu sendiri.

---

### 3. Buat Akun Vercel

Vercel adalah platform hosting gratis yang sangat cocok untuk website statis seperti HTML & CSS.

Daftar akun Vercel di sini:

👉 [https://vercel.com/signup](https://vercel.com/signup)

---

### 4. Hubungkan GitHub ke Vercel

Langkah-langkah menghubungkan GitHub ke Vercel:

1. Masuk ke dashboard Vercel
   👉 [https://vercel.com/dashboard](https://vercel.com/dashboard)
2. Klik **New Project**
3. Pilih repository GitHub hasil *fork*
4. Ikuti instruksi hingga proses deploy selesai

📚 Tutorial resmi dari Vercel:

👉 [https://vercel.com/docs/git/vercel-for-github](https://vercel.com/docs/git/vercel-for-github)

---

### 5. Selesai 🎉

Jika proses berhasil, kamu akan mendapatkan URL website online, contoh:

```
https://nama-project-kamu.vercel.app
```

Landing page kamu sekarang sudah bisa diakses secara publik.
